# Instagram-login-redesign
A mini UI redesign case study of Instagram’s login page focused on improving visual hierarchy, brand expression, and CTA emphasis through a softer Instagram-inspired interface.
## Overview
This is a UI redesign of the Instagram login page.

## Goal
To make the login screen feel more expressive, branded, and aligned with Instagram's entertainment-focused identity.

## Design Decisions
- Made the `Log in` CTA more prominent for clearer action hierarchy
- Used a soft pink gradient inspired by Instagram's brand palette
- Reduced the corporate feel of the original blue-toned interface
- Kept the layout minimal and mobile-focused

## Outcome
The redesign creates a warmer, more playful first impression while keeping the interface simple and familiar.
